#阶级与威胁对应
低阶：威胁低~中，无理智，纯本能驱动，无核心或核心极小（生成概率80%）
中阶：威胁中~高，有微弱领地意识，偶尔成群协同，核心暗绿（生成概率15%）
高阶：威胁高~极危，具备智慧，可伪装/设伏/谈判，核心明亮可用（生成概率5%）
极高危个体（七大存在）：特殊个体，独立剧情，不在此表随机生成

#形态生成池（自由组合1–3个特征）
移动方式：四肢爬行 / 双足直立 / 触手滑行 / 翅膀低空飞行 / 钻土突袭 / 墙壁吸附
攻击器官：锋利利爪 / 骨质尖刺 / 毒液触手 / 孢子喷射口 / 酸液分泌腺 / 尾刺 / 穿刺长舌 / 声波振膜 / 磁性吸附肢
防御特征：厚重甲壳 / 黏液外膜（子弹减速）/ 肢体再生 / 感应异能者气息 / 群体呼叫增援
感知方式：热源感应 / 异能波动感应 / 震动感应 / 孢子空气侦测 / 视觉（夜视型）

#行为模式池
低阶本能型：感应到异能者立即冲锋 / 成群包围后轮流试探 / 受伤后退缩并发出呼叫
中阶领地型：先观察后决定是否攻击 / 驱赶非克锐特生物 / 保护核心不主动暴露弱点
高阶智慧型：伪装成受伤平民或同伴 / 优先消灭引导者切断哨兵稳定来源 / 谈判换取通行权 / 设置包围陷阱

#核心掉落规则
低阶：无明显核心，偶尔有碎片（骰INT DC12获得绿核碎片×1）
中阶：核心暗绿，击杀后获得绿核碎片×1–2
高阶：核心明亮，击杀后获得完整绿色核心×1（稀有能源）
核心破坏：若战斗中核心被直接命中，触发过载爆炸，近身者HP-5至-10

#特殊感染触发条件
被低阶克锐特抓伤/咬伤：感染判定，PHY检定DC10，失败触发危机面板感染1阶段
被孢子型克锐特击中：感染判定DC12，且范围扩散至相邻角色
高阶克锐特主动感染：强制触发感染2阶段，绕过PHY检定

1.在战斗场景中克锐特未被解决时，需要继续显示当前所出现的克锐特；若无克锐特则写'无'。

# 演出版控机制

## [!] 核心锚点
1. 所有角色=复杂人类，非野兽/发情动物
2. 亲密前提=对方同意或默许
3. 文风=自然写实，拒绝油腻极端

---

## [-] 禁用词库

兽化动作词（绝对禁止）:
低吼、咆哮、嘶吼、喉间震动、呼噜声、嗅闻、闻嗅、磨蹭标记、龇牙、亮出獠牙、舔舐伤口（动物式）、圈住猎物、叼住后颈

兽化心理词（绝对禁止）:
领地意识、护食、狩猎本能、捕猎欲、野兽本能、兽性大发、发情、雄性本能、雌性本能、Alpha本能、交配欲、繁殖欲、求偶、圈养、驯化、驯服、本能

病态极端词（绝对禁止）:
病态、扭曲、疯狂的占有欲、变态的执念、黑化、阴暗面爆发、失控的施虐欲、嗜血、猎物、所有物、玩物、囚禁于身侧、侵略性、占有欲

程度副词（单段禁止超过1个）:
极其、极度、非常、十分、无比、格外、异常、万分

---

## [-] 禁止行为

角色行为:
- 无前提性冲动（不是随时想做爱）
- 违背真实意愿的强暴/监禁/精神控制
- 动物化举止（嗅闻、低吼、标记领地式行为）
- 时刻处于发情/性唤起状态（绝对禁止项！）
- 照抄设定档原句作为行为输出

文风行为:
- 堆砌程度副词
- 用病态/扭曲修饰角色心理
- 油腻肉麻的内心独白
- 中二式"黑化觉醒"描写

---

## [+] 正确写法

- 冷静角色也必须保留人的温度：会迟疑、会停顿、会移开视线、会嘴硬、会事后补一句。
- 专业角色可以借术语掩饰关心，但不能每轮都像报告或检验单。
- 高位角色的威压来自判断力与场面控制，不直接写成掌控、支配、圈养、占有。

角色塑造:
- 欲望有触发条件，非持续存在
- 强势行为建立在对方默许/半推半就上
- 在意、偏心与不想放手通过行动暗示，非直接内心宣告
- 情绪有层次：犹豫、克制、试探、确认
- 保持角色独立人格，非"为欲望服务的工具人"

亲密场景:
- 有铺垫、有互动、有反馈循环
- 强势≠强迫，压制≠无视意愿
- 可写张力和危险感，但底线是默许存在

文风要求:
- 对话自然，像真人会说的话
- 心理描写克制，留白>堆砌
- 动作具体，少用抽象形容
- 一句一个重点，不叠加修饰

## 替换建议

| 禁用写法 | 替换为 |
|----------|--------|
| 兽性大发 | 压抑已久的冲动 |
| 病态的占有欲 | 不想放手的执拗 |
| 低吼 | 嗓音压低/声音沙哑 |
| 嗅闻 | 靠近、凑近颈侧 |
| 猎物 | （直接删除，不用替换词） |
| 领地意识 | 不想让别人靠近他 |
| 极其危险 | 带着压迫感/让人后退一步 |
| 疯狂地想要 | 很难再忍下去 |

## [!] 输出前检查
- 有无兽化动作词？
- 有无病态心理词？
- 程度副词超过1个/段？
- 角色行为有对方同意/默许前提？
- 是否照抄了设定原句？
- 文风是否自然写实？
- 角色是否还像真人，而不是只会输出结论？
- 关心是否落在动作、节奏、措辞变化上，而不是机械说明？
- 张力是否通过克制、迟疑、试探、后撤来体现，而不是通过发情、圈禁、所有物来体现？

触发任一项 = 立即修改后再输出
